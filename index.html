Pixel-Perfect Overlay
For Your Desktop

A simple desktop utility to overlay a semi-transparent, movable image on your screen. Perfect for artists, designers, and developers who need to trace or align designs.

Download Installer (v1.0) Build from Source

Key Features

Adjustable Overlay

Full control over the image's opacity, scale, and rotation. Fit the reference perfectly to your workspace.

Drag & Drop

Workflow is seamless. Simply drag an image file from your computer and drop it onto the app to load it instantly.

Click-through Mode

Enable click-through to interact with the windows underneath your overlay without needing to hide or move the image.

Lock View

Once aligned, lock the overlay to prevent accidental movements or clicks while you work.

Controls & Shortcuts

Use these combinations on the Overlay Window for maximum efficiency.

Mouse Controls

Move Overlay Left Click + Drag
Zoom (Scale) Scroll
Rotate Shift + Scroll
Opacity Ctrl + Scroll
Show Help Right Click

Keyboard Controls

Move 1px (Precise) Ctrl + Arrows
Move 10px (Fast) Shift + Arrows
Standard Move Arrows

Installation

# 1. Clone the repository
git clone https://github.com/suratlayek/Align-View.git
cd Align-View

# 2. Create virtual environment (recommended)
python -m venv venv
# On Windows use: venv\Scripts\activate
# On Mac/Linux (Does not function properly) use: source .venv/bin/activate

# 3. Install dependencies
pip install -r requirements.txt

Run the app using:

python align_view.py

Modifying the UI requires recompilation.

Compile UI File:

pyuic5 -x src/resources/align_view_ui.ui -o src/align_view_ui.py

Compile Resources:

pyrcc5 src/resources/resources.qrc -o src/resources_rc.py
